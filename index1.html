SNCF project

How can we encourage users to favor means of transport?

App

- As part of a collaborative project between SNCF and IUT Paris Rives de Seine, I worked on implementing data science skills to promote the use of transportation. The objective was to design an innovative interactive tool in the form of a dynamic and informative website based on mobility data.

- Objective: Design an interactive website to promote the use of transportation by creatively leveraging data.

- Steps:

- Define Objectives: Identify key mobility issues and determine how data can be used to encourage the use of transportation.
- Data Collection and Analysis: Gather relevant data on travel habits, user preferences, and the performance of various modes of transportation.
- Website Development: Design a user-friendly and attractive interface using technologies such as Python to create a smooth and interactive user experience.
- Integration of Specific Features: Include features such as route planning, travel time comparisons, and real-time transportation information.

- Presentation: The project was presented to SNCF data scientists through a detailed PowerPoint presentation outlining design choices and key features.
